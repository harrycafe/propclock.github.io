Prop Clock

Prop Clock was inspired by the play 2:22. Prop Clock is a free programmable 7-segment display in a web page created using Brandon L White's SevenSeg.Js. All you need to use it is a browser and a screen. It offers:

Clock: 24 hour clock that shows the current time in hours and minutes.

SetTime: Progammable clock. Set the start and end time, set the speed of the clock, it's direction and whether a beep will be heard when the end time is reached.

For best effect, use F11 to make Prop Clock full screen on your device.

Note that beeps will not sound unless you have clicked the screen. There is a good reason for this.

Warning: Screen burn is a consideration. Extended usage could permanently damage a screen. Do not use Prop Clock as a screensaver!
